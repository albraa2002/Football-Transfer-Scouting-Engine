⚽ Live Football Transfer Scout & Player Similarity Engine

📌 Project Overview
In modern football, finding value in the transfer market requires more than traditional scouting; it demands advanced Data Science. This project is a Machine Learning-powered Scouting Engine that evaluates players across the Big 5 European Leagues to find statistically similar "Hidden Gems" for high-profile stars.

🚀 Key Business & Sporting Value
  Moneyball & ROI: Enables Sporting Directors to identify undervalued players who replicate the statistical output of €100M+ stars, saving millions in the transfer market.
  Similarity Algorithm: Implemented **Cosine Similarity** on 'Per 90' normalized metrics (xG, xAG, Progressive Passes, Progressive Carries) to generate a % match score.
  Dimensionality Reduction: Utilized **PCA (Principal Component Analysis)** to reduce complex player metrics into a 2D scouting map, clustering players with similar playstyles.
  Tactical Fingerprinting: Developed interactive Plotly **Radar Charts** to visually compare the technical footprint of the target player versus scouted targets.

🛠️ Tech Stack
Language: Python 3.x
Machine Learning: `Scikit-learn` (PCA, StandardScaler, Cosine Similarity)
Data Manipulation: `Pandas`, `NumPy`
Data Visualization: `Plotly` (Scatterpolar, Scatter), Standalone HTML/CSS Dark Sports UI
